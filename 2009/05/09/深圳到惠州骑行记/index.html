深圳到惠州骑行记

酝酿N久的骑行计划终于在五一假期间心情好的某天里完成了。 启程日不小心睡到早上9点多。。于是11点才从布吉关出发。下午6点到达，全程100多千米。 [caption id=”attachment_340” align=”alignnone” width=”300” caption=”深圳 to 惠州”][/caption] 装备: 18升背包 ， 2升水， 绳子x2，备用内胎x2，修理工具包，换洗衣物一套，雨衣，干粮若干，防风镜，头盔，地图，笔记本，医护药品，车前包&三角包…etc 据闻205国道车多人多，环境恶劣，于是稍微稍微绕了一下路： 走    布吉->横岗->坪山->坑梓->淡水->惠州 A-B  (  205国道 ) [caption id=”attachment_341” align=”alignnone” width=”450” caption=”205国道”][/caption] 人多车多灰尘多，整路段在修铁路，坑坑洼洼的。基本上没有自行车道，迫不得已只能跟汽车抢道。 短短的10多千米居然走了1个多小时。 B-C ( 整修中的横坪公路 ) [caption id=”attachment_342” align=”alignnone” width=”450” caption=”横坪公路”]********[/caption] 横坪整修得差不多了，宽大的路面&平整的柏油路，再加上车辆极少。公路车跑得十分愉快。 C-D ( 坪山-坑梓-淡水 ) 坪山内人车比较多，稍微影响骑行速度。 [caption id=”attachment_343” align=”alignnone” width=”450” caption=”白云坑”][/caption] 中午在白云坑路边休息15分钟，顺便给法拉利拍个pp [caption id=”attachment_344” align=”alignnone” width=”450” caption=”淡水-西湖酒店”][/caption] 淡水路边建设还过得去，绿化很多。当时感觉要下雨，所以没多拍pp就赶紧上路了。这里也有个西湖大酒店。 D-E ( 淡水-惠淡公路 ) [caption id=”attachment_345” align=”alignnone” width=”450” caption=”惠淡公路”][/caption] 这里很无语，路过深汕公路与惠淡公路的交互点的时候只看到有崇雅中学的指路牌，让我以为还没到，往前走了N千米看到沙田的指示牌时才发现走错路了。上边的往回走时才发现的指示牌而另外一边的路是没有的。太悲剧了。 E-F ( 惠淡公路 ) [caption id=”attachment_346” align=”alignnone” width=”450” caption=”惠淡公路”][/caption] 其实我也不知这是惠淡公路还是惠南，还是惠澳。。。一路上又没有任何标识。全路段27KM，没有自行车道。路上景色也很无聊，就这么一条路和路边的类似人行道的石头路。公路车走上边铁定扎胎。于是又得跟汽车抢道了。 [caption id=”attachment_347” align=”alignnone” width=”450” caption=”别墅”][/caption] 荒无人烟的路边居然有个度假中心。 [caption id=”attachment_348” align=”alignnone” width=”450” caption=”惠州”][/caption] 马路上有不少小石头，一路上都得小心翼翼以免扎胎。到惠州过关后走了没几米还是给扎了一下 o(╯□╰)o。 同时还发现带的备用胎拿错了，orz。幸好还带了补胎工具，不然就泪奔了。 原定计划是到汤泉泡泡温泉第二天返回的。不过由于早上启程晚了。加上路面状况不太好，到惠州时已经下午6点了。于是下榻海湖宾馆。 [caption id=”attachment_349” align=”alignnone” width=”450” caption=”海湖宾馆”][/caption] 前台MM还允许我把法拉利扛到寝室里。:D 然后晚上去市内补充水与粮食，修理材料。顺便把该见的人都见一遍。不小心又折腾到凌晨5点。 orz 。于是决定睡个懒觉第二天乘车回深圳。 [caption id=”attachment_350” align=”alignnone” width=”450” caption=”打包”][/caption] 行李+车打包后，车包貌似有点大，估计客运的长途大巴不给我上去。所以决定乘车到惠环街道上搭河源-惠州的普通长途大巴回去。30￥。 杂记： 1. 全程都是逆风骑行，郁闷得很。防风(太阳)眼镜可以大大减少眼镜干涩度，眼药水也要带上。 2 深圳貌似用法式气嘴比较多，去到HZ的时候居然到处都找不到法式气嘴的内胎或打气工具。还好某自行车店的小伙子驱车带我到一家美利达专卖店，补充大部分公路车装备(俺的车包就这里买的)。 3. 途中每隔10分钟我会补充一次水分，大概一口。有些繁忙路段由于集中精神控车忘记喝水了。晚上休息时才发现身体有点脱水(估计喝了10多杯水= =)。路上建议多带水2L+，以及安排好喝水时机。 4. 在taobao 25￥买的码表半路挂了。中途走错路有部分原因也在这里。果然便宜没好货 = = 5. 路上遇到一些车队和骑行者，路边休息时还有同道中人过来鼓励我一番 :D 。此外在路上&城市里得到不少人的帮助，感谢。
