2012 macbook pro 升级bigsur


服役9年的macbook pro用起来还是挺顺滑，打算升级到bigsur再战几年(没钱换M1 ;D)
这里用的是big-sur-micropatcher无u盘安装方案。

软件工具准备

当前系统和macbook型号

  • MacOS Catalina
  • MacBook Pro (Retina, Mid 2012)

下载补丁包

地址： http://s1.mac2x.com/macos/11.0.x/Install.macOS.Big.Sur.11.0.1.20B29.16.1.03.dmg

bigsur的镜像

地址： https://www.applex.net/pages/macos/

步骤

  1. 重启，按住Command+R不放直到启动画面进入到恢复模式

  2. 打开命令行Utilities->Terminal

    执行上边两个命令

  3. 重启正常进入系统，打开命令行

  4. 将补丁包里的Hax.dylib文件放到个人目录下

  5. 双击挂载下载好的bigsur镜像


    第一次登录后需要把wifi关闭再打开一下才会生效
